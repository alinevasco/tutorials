## [Chocolatey](https://chocolatey.org/)


### Buscar, instalar e remover

Buscar programa

	choco search evernote

Instalar programa

* SublimeText

		choco install SublimeText3

* Evernote

		choco install evernote

* MarkdownPad2

		choco install markdownpad2

Remover programa

	choco uninstall evernote
